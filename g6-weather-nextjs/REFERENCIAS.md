# 📚 Referencias del Proyecto

Este documento recopila las fuentes de información, documentación oficial, artículos académicos/técnicos y videos educativos utilizados como fundamento teórico y práctico para el diseño, desarrollo y optimización de la aplicación del clima del Grupo 6 (G6).

---

## 1. Documentación Oficial

1. **Next.js 16 (Routing & API Routes):**
   Guía oficial de Next.js sobre enrutamiento basado en la carpeta `app` y creación de endpoints de servidor (Backend-for-Frontend) para ocultar las claves de API del cliente.  
   *Enlace:* [https://nextjs.org/docs](https://nextjs.org/docs)

2. **React 19 (Hooks & Ciclo de Vida):**
   Documentación oficial de React referente al manejo de estados locales (`useState`), efectos secundarios (`useEffect`) y referencias directas a elementos del DOM (`useRef`).  
   *Enlace:* [https://react.dev/reference/react](https://react.dev/reference/react)

3. **Tailwind CSS v4 (Theme & Custom Properties):**
   Especificaciones del nuevo motor CSS v4, compilación JIT (Just-In-Time) e integración nativa de directivas y utilidades de estilo rápido.  
   *Enlace:* [https://tailwindcss.com/docs](https://tailwindcss.com/docs)

4. **MDN Web Docs (Fetch API & Asincronismo):**
   Referencia oficial de Mozilla sobre el funcionamiento del método nativo `fetch` de JavaScript para solicitudes HTTP con promesas.  
   *Enlace:* [https://developer.mozilla.org/es/docs/Web/API/Fetch_API](https://developer.mozilla.org/es/docs/Web/API/Fetch_API)

5. **TypeScript Compiler & Language Handbook:**
   Documentación oficial de TypeScript para tipado estricto e interfaces estructuradas de respuestas climáticas complejas.  
   *Enlace:* [https://www.typescriptlang.org/docs/](https://www.typescriptlang.org/docs/)

6. **Vercel Project Environment Variables Guide:**
   Guía oficial para la inyección de secretos de producción de manera segura en plataformas PaaS.  
   *Enlace:* [https://vercel.com/docs/projects/environment-variables](https://vercel.com/docs/projects/environment-variables)

---

## 2. Documentación de OpenWeatherMap

7. **OpenWeatherMap Current Weather Data API:**
   Especificación detallada de parámetros, unidades (métrica/imperial), códigos de error e íconos dinámicos provistos para las consultas del clima actual.  
   *Enlace:* [https://openweathermap.org/current](https://openweathermap.org/current)

8. **OpenWeatherMap Geocoding API:**
   Documentación para traducción de entradas de texto libre (nombres de ciudades) a coordenadas de latitud y longitud.  
   *Enlace:* [https://openweathermap.org/api/geocoding-api](https://openweathermap.org/api/geocoding-api)

##  3. Artículos Técnicos

9. **"How to Debounce Search Input in React" (FreeCodeCamp):**
   Artículo detallado sobre cómo optimizar barras de búsqueda con retrasos dinámicos en React, evitando sobrecarga a los servidores de mapas.  
   *Enlace:* [https://www.freecodecamp.org/news/debounce-search-on-input-change-in-react/](https://www.freecodecamp.org/news/debounce-search-on-input-change-in-react/)

10. **"A Complete Guide to Glassmorphism in CSS" (CSS-Tricks):**
    Estudio detallado del diseño de interfaces de vidrio esmerilado usando filtros de CSS (`backdrop-filter`) y sombras dinámicas para el contraste del texto.  
    *Enlace:* [https://css-tricks.com/glassmorphism-css-monitor/](https://css-tricks.com/glassmorphism-css-monitor/)
